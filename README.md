# La Farine Patisserie & Café

A modern, elegant bakery and café website built with React, Vite, TailwindCSS, and Spline 3D.

## About

La Farine Patisserie & Café is a premium bakery website showcasing artisan French pastries and café offerings. The design embodies warmth, elegance, and the handcrafted nature of traditional French baking.

**Location:** Shraddhanand Chowk, opposite A Square Building, Laxminagar, Nagpur 440022  
**Phone:** +91 88067 54042  
**Cost for Two:** ₹350 approx

## Tech Stack

- **React 18** - UI Framework
- **Vite** - Build Tool & Dev Server
- **TypeScript** - Type Safety
- **TailwindCSS** - Utility-First Styling
- **Framer Motion** - Smooth Animations
- **Spline 3D** - Interactive 3D Elements
- **Shadcn/UI** - Component Library
- **React Router** - Client-Side Routing

## Features

### Interactive 3D Elements
- Spline 3D integration for bakery item models
- 360° rotation capability on 3D models
- Fallback animated illustrations when Spline scenes aren't available

### Responsive Design
- Mobile-first approach
- Tablet-optimized layouts
- Desktop-beautiful presentations
- Fluid typography and images
- Smooth breakpoint transitions

### Complete Menu System
- 21+ menu categories
- 80+ menu items
- Real-time search filtering
- Category-based navigation
- Bestseller highlighting

### Pages
- **Home** - Hero, features, bestsellers, about, gallery, contact
- **Menu** - Complete menu with search and filter
- **About** - Company story, values, statistics
- **Contact** - Contact form, info cards, embedded map

### Animations
- Scroll-triggered fade animations
- Hover effects on menu cards
- Floating elements
- Smooth page transitions
- Parallax effects

### UI Components
- Sticky navigation bar
- Floating order button (mobile)
- Toast notifications
- Glass-morphism cards
- Custom button variants

## Design System

### Color Palette
- **Primary:** Rich Coffee Brown
- **Accent:** Golden Caramel
- **Secondary:** Warm Cream
- **Highlight:** Soft Pink/Blush
- **Background:** Warm White

### Typography
- **Display:** Playfair Display (serif)
- **Body:** Poppins (sans-serif)

### Custom Shadows
- Soft shadows for subtle depth
- Medium shadows for cards
- Elevated shadows for modals
- Glow effects for CTAs
